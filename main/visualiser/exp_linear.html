Mathematics Behind Linear Regression

The equation of regression line is represented as:


h(xi) = ß0 + ß1xi


Here,

  • h(xi) represents the predicted response value for ith observation
  • ß0 and ß1 are regression coefficients and represent y-intercept and slope of regression line respectively.


To create our model, we must “learn” or estimate the values of regression coefficients ß0 and ß1. And once we’ve estimated these coefficients, we can use the model to predict responses!

In this article, we are going to use the principle of Least Squares. Now consider:


yi = ß0 + ß1xi + εi ➩ εi = yi - h(xi)


Here, εi is a residual error in ith observation. So, our aim is to minimize the total residual error. We define the squared error or cost function, J as:


J(ß0, ß1) = (1/2n)∑ni=1 ε2i


and our task is to find the value of ß0 and ß1 for which J(ß0,ß1) is minimum! Without going into the mathematical details, we present the result here:


ß1 = SSxy/SSxx


ß0 = ymean - ß1xmean

SSxy = ∑(xi - xmean)(yi - ymean)
